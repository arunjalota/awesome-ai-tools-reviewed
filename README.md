## How tools earn the daily-driver badge

A tool gets ⭐⭐⭐⭐⭐ on this list only if I'd be genuinely unhappy if it disappeared tomorrow. I review on:

1. **Output quality** — would I ship what it produced?
2. **Speed** — does it make me 2x+ faster on a real task?
3. **Reliability** — same prompt twice, same quality?
4. **Cost** — does the price match the value at my volume?
